HRU-485-X-I is a data transmission device based on industrial grade LoRa modulation technology, with strong transmission performance and long-distance transmission capability. This device supports point-to-point, point-to-many, and multi-level relay network configurations, and can achieve various data transmission methods. At the same time, the device supports data encryption and communication key functions, effectively preventing data interception. The device also has perfect shielding design and strong anti-interference ability, which can stably transmit data in harsh environments. In addition, the device also supports 4G and Ethernet (DTU01/02), with features such as low power consumption, easy installation and configuration. All parameters can be programmed, such as power, frequency, rate, address ID, etc. This device can be widely used in various industrial scenarios, meeting the data transmission needs of different scenarios, and providing users with more convenient and efficient data transmission methods.

**HRU-485-X-I have following main features:**

- ESP32 + SX1262;
- RS-485 bridge.
