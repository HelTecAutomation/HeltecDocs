HRI-485x is a data transmission device based on industrial grade LoRa modulation technology, with strong transmission performance and long-distance transmission capability. This device supports point-to-point, point-to-many, and multi-level relay network configurations, and can achieve various data transmission methods. At the same time, the device supports data encryption and communication key functions, effectively preventing data interception. The device also has perfect shielding design and strong anti-interference ability, which can stably transmit data in harsh environments. In addition, the device also supports 4G and Ethernet , with features such as low power consumption, easy installation and configuration. All parameters can be programmed, such as power, frequency, rate, address ID, etc. This device can be widely used in various industrial scenarios, meeting the data transmission needs of different scenarios, and providing users with more convenient and efficient data transmission methods.

**HRI-485x have following main features:**

+   MCU:ESP32C3-FN4 / ESP32DOWDQ6;
+   LoRa:SX1262;
+   Support 4G and Ethernet;
+   Support 4.5-28v supply voltage ;
+   Support for data encryption,1024 bytes cache,a single packet supports up to 240 bytes;
+   Support communication key function , effectively prevent data interception;
