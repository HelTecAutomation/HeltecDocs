This document branch is for `Heltec ESP32 + Arduino Node series <https://heltec.org/proudct_center/esp-arduino/>`_,

ESP32 is a powerfull SoC with 240MHz Tensilica LX6 dual-core & 1 ULP core, 600 DMIPS, 520KB SRAM. The benefit of "ESP32 + Arduino" combination:

-  Multi wireless communication technical on board, include Wi-Fi, BlueTooh/BLE;
-  Powerful computing skills;
-  Large FLASH size;
-  Power mananagement system and Li-Po support;
-  Arduino-compitable;
-  We also provide `ESP32 Node Arduino library <https://github.com/HelTecAutomation/Heltec_ESP32>`_;
