Universal E-Ink Driver
======================
:ht_translation:`[简体中文]:[English]`

This is a universal driver board for E-Ink, connects via FFC and is compatible with most FPC interfaces for E-Ink. It supports WiFi, Bluetooth to refresh, We have uploaded the routine that refresh via web page and serial port yet. 
It can also be used as an Arduino development board specifically for Eink drivers, we provide the development environment and its circuit schematic, you can use it for corresponding development work.
