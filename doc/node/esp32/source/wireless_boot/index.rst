Heltec WirelessBoot System
==========================
:ht_translation:`[简体中文]:[English]`

Wireless Boot is a boot loader system for ESP32 MCU. It is used for devices that do not have an external USB or serial port. You can download firmware, exchange information, and print logs through WiFi. Therefor, devices no longer requires a USB-UART bridge, which can reduce the hardware cost of the system and reduce device size.
