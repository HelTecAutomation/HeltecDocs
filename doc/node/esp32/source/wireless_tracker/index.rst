Wireless Tracker
================
:ht_translation:`[简体中文]:[English]`

Wireless Tracker is a development kit based on ESP32-S3FN8. It integrates both
SX1262 and UC6580 to provide fast GNSS solution for IoT. It can track any object and
then upload that data wirelessly by Wi-Fi, Bluetooth, LoRa. Wireless Tracker supports L1 + L5/L2, 
and supports GPS, GLONASS, BDS, Galileo, NAVIC, QZSS multi-system joint positioning.

**WiFi-LoRa-32 Quick Start (development framework and special libraries)**, please refer to `Heltec ESP32 series Quick Start <https://docs.heltec.org/en/node/esp32/quick_start.html>`_.

**LoRaWAN example** please refer to: `Heltec ESP32 series LoRaWAN example <https://docs.heltec.org/en/node/esp32/lorawan/index.html>`_.

**Meshtastick Quick Start**, please refer to this link: `Meshtastick Quick Start <https://docs.heltec.org/en/node/esp32/meshtastick.html>`_.

**Download related resources**, including schematics, pin diagrams, and data sheets: `https://resource.heltec.cn/download/Wireless_Tracker <https://resource.heltec.cn/download/Wireless_Tracker>`_.
